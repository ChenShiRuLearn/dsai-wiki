## 下载

### Docker

```
docker pull apachecn0/dsai-wiki
docker run -tid -p <port>:80 apachecn0/dsai-wiki
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install dsai-wiki
dsai-wiki <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dsai-wiki
dsai-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
